== Description ==
Mit dem Analytics-Gtag kann man den neuen Global Tag von google sowie Facebook hinzufügen. Desweiteren erlaubt es folgende zusatzfunktionen.

1. Ip Annonymisierung (die in Deutschland und Oesterreich plicht ist)
2. Optout-Ffunktion die in dem Header hinzugefügt wird
3. Deaktivierung des trackings
4. Youtube tracking
5. Scroll Tracking oder Erweiterstes Scrolltacking
6. Kontaktform 7 tracking


== Installation ==
Upload the entire Analytics-gtag folder to the /wp-content/plugins/ directory.
Activate the plugin through the ‚Plugins‘ menu in WordPress.
You will find ‚Contact‘ menu in your WordPress admin panel.

For basic usage, you can also have a look at the plugin web site.
